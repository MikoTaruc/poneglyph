# Rsbuild React App

A minimal React application scaffolded for Rsbuild.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Preview the production bundle:
   ```bash
   npm run preview
   ```

## Notes

- This project uses `@rsbuild/core` and `@rsbuild/plugin-react`.
- Make sure your local Node version meets Rsbuild requirements (Node 22+ is recommended).
